# MEMORY
Memory - это приложение по тестированию памяти.

![image_1](image_1.png)

## Getting started
```npm install``` <br>
```nodemon or npm start```

## MVP Functionality
- запоминание произносимых слов в указанном порядке
- заполнение слов в порядке их воспроизведения
- вывод результатов тестирования на экран
